# Matemática Sem Mistérios

## Sobre o projeto

O **Matemática Sem Mistérios** é um blog criado para compartilhar conhecimentos e dicas que ajudam a aprender matemática de forma simples e sem complicações.

Este projeto foi desenvolvido como atividade prática da Alura, utilizando HTML e CSS.

## Frase do blog

> "VOU COMPARTILHAR CONHECIMENTOS PARA DESCOMPLICAR A MATEMÁTICA"

## Conteúdo do primeiro post

**Meu Primeiro Post**
Por Gabrieli Gubert

Boas-vindas ao meu blog! Aqui você irá aprender dicas para aprender matemática rapidamente e sem complicações.

Meu objetivo é mostrar que a matemática pode ser fácil, divertida e acessível para todos.
